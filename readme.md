# redui-icons

redui icons

矢量 icon 



### 参数 (Props)
| 参数名  | 说明      | 必填   | 类型       | 默认值     | 可选值                                      |
| ---- | ------- | ---- | -------- | ------- | ---------------------------------------- |
| type | icon类型  | 否    | `string` | success | alert,prompt,wait,tick,close,warning,success |
| size | icon 大小 | 否    | `string` | small   | small, large                             |
